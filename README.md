### Basic Commands

cd - Change Folder

type - Print file contents

create - Create New File

del - Delete File

mkdir - Create New Folder 

rmdir - Delete Folder

dir - Print contents of current folder

cls - Clear Console

help - Print summary of list of commands or single command syntax + summary
